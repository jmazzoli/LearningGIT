Learning GIT

Primeros comandos

  • git init
  • git add -A
  • git status
  • git commit -m "message"
  • git log

Branch

  • git branch nombre
  • git ckeckout branch name
  • git merge branch, posicionado en el master
  • git branch sin parámetro, lista las ramas
  • git branch -a lista también las ramas ocultas

GitHub

  • git remote add origin url
  • git remote -v
  • git push origin branch
  • issues, milestones, labels

Tags

  • git commit --amend -m "nuevo mensaje en el commit"
  • git push origin branch -f fuerza el commit cuando no hubo cambio en el código
  • git tag -a v1.0 -m mensaje hash de un commit específico
  • git push origin v1.0
  • git push origin --tags
